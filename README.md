# 📖 Bionic Reading Userscript (仿生阅读 油猴脚本)

一个通用的浏览器油猴脚本，用于在英文网页上应用 **仿生阅读 (Bionic Reading)** 效果，加粗单词的起始部分，帮助用户更快地定位和识别文本。

这个版本解决了旧版脚本中存在的**DOM 暴力替换**和**无法无刷新恢复**的问题，提供了稳定且高效的用户体验。

## ✨ 特性 (Features)

* **默认启动 (Default Enabled):** 脚本在页面加载完成后自动启用，无需手动操作。
* **非破坏性修改 (Non-Destructive):** 使用 DOM 标记和文本缓存实现修改和恢复，不依赖页面刷新。
* **稳定切换:** 通过新的快捷键实现快速、可靠的开关，解决浏览器冲突。
* **智能加粗:** 根据单词长度和常用词字典动态调整加粗字母的数量。

## ⚙️ 核心逻辑 (Technical Details)

本脚本的核心在于采用了**非破坏性的修改**策略：

1.  脚本遍历 `TextNode`。
2.  它不会直接替换 `TextNode`，而是将原始文本内容存储在一个新的 `<span>` 元素的 `data-bionic-original` 属性中。
3.  用这个包裹了格式化 HTML 的 `<span>` 替换原始 `TextNode`。
4.  当用户禁用功能时，脚本查找所有带有 `data-bionic-original` 属性的 `<span>`，提取原始文本，并用一个新的原始 `TextNode` 替换回 `<span>`，从而实现无刷新恢复。
